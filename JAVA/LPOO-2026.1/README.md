Testando a ligação de um projeto Maven feito com JavaFX e Scene Builder, com um banco da dados em MySQL.

Dependências usadas:
    1. org.hibernate.orm  hibernate-core  versão 7.3.0.Final
    2. org.projectlombok  lombok  versão 1.18.44
    3. com.mysql  mysql-connector-j  versão 9.6.0
    4. org.openjfx  javafx-controls  21
    5. org.openjfx  javafx-fxml  21

Utilização de: Modelo MVC, DAO, persistence.xml e arquivos FXML para agirem como Views.
